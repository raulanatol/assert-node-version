# Assert node version

Assert is the node version it's correct in your repositories

## How to install?

```shell
npm install -g assertnv
```

Or if you don't like install

```shell
npx assertnv
```

## How to use

To create a repository configuration (only the first time in your repo):

```shell
assertnv init
```

## How to check

```shell
assertnv
```

or

```shell
assertnv check
```
